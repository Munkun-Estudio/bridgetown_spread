# Bridgetown Development Practices

This repository serves as a centralized hub for our company's best practices, guidelines, and standards for web development with [Bridgetown](https://www.bridgetownrb.com/).

## Purpose

The main goal of this repository is to document and share our collective knowledge to ensure consistency, quality, and efficiency across all our Bridgetown projects and to serve as a reference for new team members. We also hope to help others know about Bridgetown and how to use it effectively.

This repository will include:

- ~~AI assistant configuration~~
- Code style guides
- Recommended plugins and configurations
- Component architecture patterns
- Deployment procedures
- Performance optimization tips
